2022-1 프로그래밍랩 기말 대체 개인 텀프로젝트

1. 제목 : 무한의 계단
2. 프로젝트 완성도
	(1) 완성도 : 70%
	(2) 아쉬운점 : 코인이나 속도제어 아이템을 다양하게 하지 못함
3. 기능 : 
	(1) 1인 Play (방향키 이용)
	(2) 계단이 랜덤으로 무한개
	(3) 시간 제한
	(4) 계단에서 떨어지면 게임 오버
